### OVERVIEW
WooCommerce Cross Sell Products Display Plugin provide you the option to display your woocommerce cross sell products using easy to use widgets and shortcodes. 

### How it works?  
1. After successfull plugin activation go to your wordpress admin `Widgets` page.  
2. If you are in admin `Widgets` page, you can find a widget named **Woocommerce Cross Sells Product Display**.  
3. Drag this **Woocommerce Cross Sells Product Display** widget inside your desired sidebar then fill the widget fields and save.  
4. Now check the front end.  
5. You can also use shortcode to display Cross Sells Products. For this use the shortcode in any text widget, in post content, in page content or using php code <?php echo do_shortcode('[cdxwoocs pid="101" theme="theme-list-view"]'); ?>. 

### SHORTCODE  
You can also use [cdxwoocs] shortcode to display woocommerce cross sell products. 
This shortcode has two option/attribute 
**1. pid** - It is a mandatory attribute of the shortcode and it denotes to prodcut id of which you want to the cross sell products. 
**2. theme** - It denotes to display layout. Use "theme-list-view" or "theme-hover" as its value.

### SHORTCODE Example    
[cdxwoocs pid="101" theme="theme-list-view"]

### IMPORTANT NOTICE  

- Plugin works based on WooCommerce plugin. So WooCommerce plugin must be installed and active for this plugin to work.
- Plugin - Widget works only on product details page as it shows the cross sell products(if any) of the current product.  
- If you want to display cross sell products any page/post then use shortcode with proper option.
- Target woocommerce product must have Cross-sells products set to display.
- If you want to know how to setup woocommerce Cross-sell products try this [youtube link &raquo;](https://www.youtube.com/results?search_query=cross+sell+products+woocommerce "Codextent - Get Best Wordpress Plugins & Extensions") with many tutorials about it 

== Installation ==

1. Unzip and upload the `cross-sell-product-display-for-woocommerce` directory to the plugin directory (`/wp-content/plugins/`) or install it from `Plugins->Add New->Upload`.  
2. Activate the plugin through the `Plugins` menu in WordPress.  
3. That's all you need to do. You will notice a new **Woocommerce Cross Sells Product Display** widget in your wordpress admin widgets page.
